# Travel Agency Website

## Description
This is a **Travel Agency Website** that showcases popular travel destinations in India such as **Delhi, Goa, Mumbai**, and more. The website provides users with an interactive and visually appealing interface to explore destinations, view images, and read brief information about each city. The project focuses on **frontend development** using HTML, CSS, and JavaScript, with engaging hover effects and responsive design.

## Features
- **Destination Cards:** Each city has a card with an image, title, and short description.
- **Interactive Hover Effects:** On hover, the card highlights the heading, image, and information for a more engaging user experience.
- **Responsive Design:** The website works well on desktops, tablets, and mobile devices.
- **Dynamic Layout:** Grid-based layout for organized presentation of multiple destinations.
- **Clean UI:** Modern and minimalistic design for better readability and user experience.

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Responsive Design:** CSS Flexbox/Grid, Media Queries
- **Styling:** Hover effects, transitions, and shadows for interactive elements

## How It Works
1. Users open the website in a browser.
2. The homepage displays a grid of destination cards (Delhi, Goa, Mumbai, etc.).
3. Hovering over a card highlights it and shows additional information.
4. Users can explore multiple destinations easily through the responsive interface.
